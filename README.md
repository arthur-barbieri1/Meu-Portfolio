## ⚙️ Quer rodar na sua máquina?

Só seguir o passo a passo:

bash
# Clonar o projeto
git clone https://github.com/arthur-barbieri1/portfolio.git

# Entrar na pasta
cd portfolio

# Instalar as dependências
npm install

# Rodar o projeto
npm run dev
