## 💻 Quer rodar na sua máquina?

Só seguir o passo a passo:

```bash
# Clonar o projeto
git clone https://github.com/arthur-barbieri1/Meu-Portifolio.git

# Entrar na pasta
cd Meu-Portifolio

# Instalar as dependências
npm install

# Rodar o projeto
npm run dev


Acesse http://localhost:5173 no seu navegador.
